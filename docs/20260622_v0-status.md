## ✅ Features implémentées

### Core algorithme

- `useVentilationScore` : ΔT (ressenti intérieur − ressenti extérieur) + `uvPenalty` + bonus nocturne
- `getBestVentilationHour` : fenêtre glissante [now, now+24h], heures futures uniquement
- Seuil `isFavorable > 2°C` pour conseiller l'aération
- `ComfortLevel` 3 niveaux (hot/neutral/cool) avec biais additif sur le delta

### Données météo

- `useWeatherForecast` : appel Open-Meteo sans clé API
- Variables : `temperature_2m`, `apparent_temperature`, `relativehumidity_2m`, `windspeed_10m`, `uv_index`
- `forecast_days=2` + filtre [now, now+24h] pour traverser minuit
- `sunrise` / `sunset` J et J+1

### Géolocalisation

- `useLocation` : GPS natif + fallback recherche commune
- `LocationSearch` : autocomplete API Adresse gouvernement
- Source discriminée (`"gps"` vs `"search"`) dans le type `GeoLocation`

### UI — Composants

| Composant             | Description                                                                        |
| --------------------- | ---------------------------------------------------------------------------------- |
| `RecommendCard`       | Orchestrateur — contient les 4 sous-composants                                     |
| `VerdictBanner`       | Verdict unifié sur `score.score` avec 4 états (good/neutral/wait/bad)              |
| `IdealSlots`          | Créneaux idéaux gros chiffres + icône `TIME_OF_DAY_META` + label jour conditionnel |
| `ThermalComparison`   | Comparaison thermique int/ext                                                      |
| `ThermalDelta`        | Affichage du ΔT avec couleur seuil                                                 |
| `VentilationTimeline` | Chart bidirectionnel barres ± score, 24h                                           |
| `CalendarLink`        | Génère et télécharge `.ics` côté client                                            |
| `TipsSection`         | Carousel 10 conseils canicule, swipe + auto-défilement + haptics                   |
| `LocationSearch`      | Autocomplete commune avec GPS fallback                                             |

### Infrastructure

- `localStorage` : persistance `indoorTemp` + `comfortLevel`
- `useHaptics` : wrapper `web-haptics` avec mapping UX par zone
- `LazyMotion` + `AnimatePresence` : animations entrée + transitions état
- Palette warm orange (CSS vars `--ember`, `--verdict-*`)
- `manifest.json` + `<link rel="manifest">` dans `index.html`
- `favicon.svg` (ThermometerSun Lucide)
- Mobile-first, `max-w-sm` centré
