# BDR-060 — App state vigilance : état global différé V1

| Décision                                                                                            | Pourquoi                                                                                                                  | Alternatives considérées                             | Statut |
| --------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- | ------ |
| Banner conditionnelle simple en V0 ; app state (bg + titres colorés selon vigilance max) différé V1 | Transformation UI profonde — fond de page + titres virant à la couleur de vigilance max mérite son propre cycle de design | App state immédiat en V0 (rejeté : scope trop large) | actif  |

## Notes

- V0 : `VigilanceBanner` visible uniquement si au moins une alerte active (color ≠ "vert")
- V1 : exposer `highestColor` depuis `useVigilanceData`, propager via context ou store, adapter `bg`, `text-*`, animations globales selon le niveau
- Priorité couleur : rouge > orange > jaune > vert
