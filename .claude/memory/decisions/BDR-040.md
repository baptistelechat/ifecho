# Footer → page Politique de confidentialité RGPD

PostHog activé avec Session Replay + Product Analytics + Error Tracking.
Avant déploiement public, ajouter un lien "Confidentialité" dans le footer de l'app
pointant vers une page dédiée qui explique :

- Données collectées : events anonymisés, sessions replay sans données perso
- Aucun cookie persistant (`persistence: 'memory'`)
- Hébergement EU (eu.posthog.com) — RGPD compliant
- Masquage automatique des champs texte (Session Replay)

**Why:** Session Replay nécessite un minimum de transparence envers les utilisateurs, même sans données perso. Requis légalement (RGPD Art. 13/14).

**How to apply:** Créer une route `/privacy` ou `/confidentialite` dans l'app Vite + React. Contenu minimaliste (10-15 lignes). Lien dans `AppFooter` à côté des autres liens existants.
