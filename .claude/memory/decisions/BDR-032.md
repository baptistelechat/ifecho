# BDR-032 — Pas de spinner météo : `hasContent` conditionnel

| Décision                                                                                                                      | Pourquoi                                                                                                                         | Alternatives considérées           | Statut |
| ----------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------- | ------ |
| `hasContent = weather && !isLoadingWeather && location` conditionne tout le bloc contenu — rien affiché pendant le chargement | Un spinner météo intermédiaire crée un décalage visuel : la page passe de spinner → card, provoquant un layout shift perceptible | Spinner `Loader2`, skeleton loader | actif  |
