# BDR-016 — Édition inline température : `isEditing` toggle + conteneur `h-[1em]`

| Décision | Pourquoi | Alternatives considérées | Statut |
| --- | --- | --- | --- |
| Clic sur le chiffre déclenche un `<input>` inline via toggle `isEditing`. Conteneur `div.flex.h-[1em]` fixe la hauteur à 1em (= `text-5xl`), `h-full` sur l'input hérité. Enter/blur valide, Escape annule. | Confort UX : édition immédiate sans slider obligatoire. Layout shift évité : `<input>` intrinsic height > `<button>` même avec `p-0 border-0` — seul un conteneur à hauteur fixe l'absorbe. | Input permanent visible (dégrade le rendu lecture) ; champ séparé en bas (couvert par le slider existant) | actif |
