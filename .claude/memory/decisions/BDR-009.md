# BDR-009 — Palette warm orange (#fff8f4 + ember #ea580c) comme identité visuelle

| Décision                                                                                      | Pourquoi                                                                                                                       | Alternatives considérées                                                     | Statut |
| --------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------- | ------ |
| Background crème (#fff8f4) + accent ember orange (#ea580c) remplacent la palette bleue froide | L'app représente la canicule/chaleur — une palette froide dissociait l'UI de sa sémantique et évoquait un contenu IA générique | Dark mode bleu/gris (rejeté par Baptiste), palette bleue/sky, neutre système | actif  |

## Exception intentionnelle

Les icônes Cloud et Droplets dans `ThermalComparison` gardent `text-sky` : contraste sémantique "air extérieur frais vs intérieur surchauffé". Bleu = cool dehors, orange = chaud dedans.
