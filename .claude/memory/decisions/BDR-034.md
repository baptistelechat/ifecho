# BDR-034 — `@vite-pwa/assets-generator` + `logo.svg` source unique pour icônes PWA

| Décision                                                                                                                           | Pourquoi                                                                                                                                         | Alternatives considérées                     | Statut |
| ---------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------- | ------ |
| Utiliser `@vite-pwa/assets-generator` avec un script pnpm `generate-pwa-assets` régénérable, depuis `logo.svg` comme source unique | Génère tous les formats requis (pwa-64, pwa-192, pwa-512, maskable-512, apple-180, favicon.ico) en une seule commande ; rejouable si logo change | Script bash/ImageMagick manuel, export Figma | actif  |
