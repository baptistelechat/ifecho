# BDR-067 — `bulletinStale` global sur agrégat de tous les items J

| Décision                                                                                         | Pourquoi                                                                                                                                                  | Alternatives considérées                         | Statut |
| ------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ | ------ |
| Calculer `bulletinStale` en agrégeant TOUS les items J de tous les phénomènes, avant le `.map()` | Per-phénomène rate le cas `j=[]` : un phénomène vert hier n'a pas d'item J, donc `jAllExpired` reste false même quand le bulletin est périmé après minuit | `jAllExpired` per-phénomène (rejeté — incomplet) | actif  |

## Détail

```tsx
const allJItems = [...groups.values()].flatMap((g) => g.j);
const bulletinStale =
  allJItems.length > 0 &&
  allJItems.every((item) => new Date(item.end_time) <= now);
```

Ensuite dans le map, pour chaque phénomène :

```tsx
const todayItems = bulletinStale ? j1 : j;
const tomorrowItems = bulletinStale ? [] : j1;
```
