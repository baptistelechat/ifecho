# BDR-023 — Fenêtre glissante `[now, now+24h]` pour timeline et créneaux idéaux

| Décision                                                                                               | Pourquoi                                                                                              | Alternatives considérées                                 | Statut |
| ------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------- | -------------------------------------------------------- | ------ |
| Afficher toujours les 24 prochaines heures depuis l'instant présent, pas la journée calendaire 00h–23h | Une fenêtre fixe calendaire tronque les créneaux traversant minuit (ex. "22h–06h" devenait "22h–00h") | Journée calendaire 00h–23h — écartée : créneaux tronqués | actif  |

Implémentation : `forecast_days: "2"` pour couvrir J et J+1, filtre `date === nowDate && hour >= nowHour` OU `date > nowDate && hour < nowHour`. Séparateur visuel (ligne pointillée + date DD/MM) à la frontière J/J+1 dans VentilationTimeline.
