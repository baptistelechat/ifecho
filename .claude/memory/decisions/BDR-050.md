# BDR-050 — UTM params injectés dans l'URL de partage

| Décision                                                                                                                             | Pourquoi                                                                                                                                                                                                                                | Alternatives considérées | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------ | ------ |
| `buildShareUrl(method)` injecte `utm_source=share`, `utm_medium=native\|clipboard`, `utm_campaign=viral` dans l'URL avant le partage | Sans UTM, le trafic viral in-app est indiscernable du trafic direct dans PostHog. Le `method` dans `utm_medium` distingue le partage natif mobile du copier-coller desktop. Une URL nue (sans UTM) ne crée aucun tracking d'acquisition | URL nue sans paramètres  | actif  |
