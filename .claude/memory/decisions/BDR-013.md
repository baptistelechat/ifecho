# BDR-013 — Centralisation styles shadcn : variants dédiés, 0 className ad-hoc

| Décision                                                                                                     | Pourquoi                                                                                              | Alternatives considérées                       | Statut |
| ------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------- | ---------------------------------------------- | ------ |
| Tous les styles UI passent par les variantes CVA des composants shadcn — pas de className de style à l'usage | Un className résiduel sur un composant shadcn signale une variante manquante dans le composant source | Styler chaque usage directement avec className | actif  |

Variants créés dans `button.tsx` : `surface` (bouton carte, border + hover ember), `cta` (plein-largeur, rounded-2xl), size `icon` (rounded-full h-10 w-10). Styles absorbés dans `input.tsx` : `border-border`, `bg-input`, `rounded-xl`, `py-2.5`, focus ring subtil. Slider shadcn installé via `@radix-ui/react-slider` pour remplacer `<input type="range">`.
