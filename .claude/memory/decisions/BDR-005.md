# BDR-005 — Double précision météo : commune UI vs centroïde département RPi

| Décision                                                                                                                              | Pourquoi                                                                                                                                                | Alternatives considérées                                                             | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ | ------ |
| UI affiche météo exacte de la commune (lat/lon API Adresse) ; notifs RPi utilisent centroïde département (`src/data/departements.ts`) | Scalabilité fixe pour le RPi (≤ 101 fetches/heure), meilleure expérience UI. Trade-off accepté : notif basée sur le département, pas la commune exacte. | Commune exacte pour les deux (non scalable), département pour les deux (UX dégradée) | actif  |
