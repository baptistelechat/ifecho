# BDR-045 — `device_type` custom retiré → `$os` natif PostHog

| Décision                                                                                                             | Pourquoi                                                                                                                                             | Alternatives considérées                                                | Statut |
| -------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- | ------ |
| Supprimer `device_type: getDeviceType()` de `analytics.appOpened()` dans App.tsx, s'appuyer sur `$os` de `$pageview` | PostHog capture nativement `$os` sur chaque `$pageview` — doubler manuellement est redondant et source d'incohérence possible entre les deux valeurs | Garder `device_type` : écarté (valeur identique à `$os`, code superflu) | actif  |
