# BDR-036 — `logo.svg` source génération uniquement ; `favicon.ico` favicon déployé

| Décision                                                                                                            | Pourquoi                                                                                                                                           | Alternatives considérées                                   | Statut |
| ------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- | ------ |
| Renommer `favicon.svg` → `logo.svg` ; `favicon.ico` généré par `@vite-pwa/assets-generator` devient le vrai favicon | Séparation claire des rôles : chaque fichier porte le nom de son usage réel ; `logo.svg` = source de génération réutilisable pour d'autres formats | Garder `favicon.svg` comme source et favicon simultanément | actif  |
