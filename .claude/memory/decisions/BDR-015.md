# BDR-015 — Seuil `isFavorable > 2°C` pour conseiller l'aération

| Décision                               | Pourquoi                                                                                                       | Alternatives considérées                                                     | Statut |
| -------------------------------------- | -------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- | ------ |
| Score ventilation favorable si `> 2°C` | Un gain de <2°C de ressenti est marginal — ouvrir les fenêtres pour 0.5° ne vaut pas la perturbation thermique | `> 0` (trop sensible aux décimales), `> 3` (trop restrictif journées douces) | actif  |
