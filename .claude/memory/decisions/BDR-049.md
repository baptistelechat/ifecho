# BDR-049 — ShareButton texte universel constant

| Décision                                                                                                                                         | Pourquoi                                                                                                                                                                                                                | Alternatives considérées                                                                         | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ | ------ |
| Un seul `SHARE_TEXT` constant universel : "Ifecho m'a conseillé d'aérer mon logement aujourd'hui 🌡️ Et toi, tu sais quand ouvrir tes fenêtres ?" | Logique conditionnelle 3 branches (ville+heure / ville seule / générique) = complexité sans gain — le message générique est plus honnête (partage l'app, pas un conseil ponctuel) et fonctionne dans tous les contextes | Texte dynamique enrichi avec ville + heure optimale ("Ifecho me conseille d'aérer à 6h à Paris") | actif  |
