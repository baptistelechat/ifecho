# BDR-035 — `vite-plugin-pwa` : `manifest: false` + workbox minimal pour SW

| Décision                                                                                                          | Pourquoi                                                                                                                                | Alternatives considérées                                                 | Statut |
| ----------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ | ------ |
| `registerType: "autoUpdate"`, `manifest: false`, `globPatterns: ["**/*.{js,css,html,svg}"]`, `runtimeCaching: []` | `manifest: false` préserve le `public/manifest.json` custom (contrôle icônes et métadonnées) ; workbox minimal suffit pour SPA statique | Génération manifest automatique par le plugin, stratégies cache avancées | actif  |
