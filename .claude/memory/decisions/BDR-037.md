# BDR-037 — Bouton d'installation PWA in-app : `useInstallPrompt` + `InstallButton`

| Décision                                                                                                                                                                                                                                                                                                                 | Pourquoi                                                                                                                                                                                                                                    | Alternatives considérées                                                  | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- | ------ |
| Hook `useInstallPrompt` capture `beforeinstallprompt` + détecte iOS/standalone. `InstallButton` positionné en `absolute top-4 right-4` dans le header (`relative`). Android → `deferredPrompt.prompt()` direct. iOS → tooltip avec instruction Share + "Sur l'écran d'accueil". Invisible si `display-mode: standalone`. | L'utilisateur peut ignorer le prompt système au premier chargement — le bouton in-app lui permet de revenir à l'installation plus tard sans relancer le site. L'`appinstalled` event supprime le bouton automatiquement après installation. | Prompt système seul sans rattrapage ; bannière persistante en bas de page | actif  |
