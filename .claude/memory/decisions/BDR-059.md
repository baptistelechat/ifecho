# BDR-059 — `analytics.ts` dans `src/lib/` (pas `src/hooks/`)

| Décision                                                  | Pourquoi                                                                                                              | Alternatives considérées                                             | Statut |
| --------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- | ------ |
| Fichier renommé `analytics.ts` et déplacé dans `src/lib/` | Module singleton sans hooks React — convention : `hooks/` réservé aux vrais hooks React (`useState`/`useEffect`/etc.) | Garder dans `src/hooks/` (nom trompeur), déplacer à la racine `src/` | actif  |
