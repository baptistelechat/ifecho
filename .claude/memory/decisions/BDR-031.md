# BDR-031 — Slide global `m.div` comme entrée du contenu principal

| Décision                                                                                      | Pourquoi                                                                                                                                                                               | Alternatives considérées                                                                    | Statut |
| --------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ------ |
| Single `m.div key="content"` avec `y: 24→0` / exit `y: -8` wrappant tout le contenu principal | `staggerChildren` et delays explicites sur chaque card sont invisibles dans ce setup LazyMotion/domAnimation — cause non identifiée. Le slide global est plus simple, fiable et propre | `staggerChildren: 0.18` sur variants, delays explicites 0/0.14/0.28/0.42 sur chaque `m.div` | actif  |
