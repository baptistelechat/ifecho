# BDR-038 — Sous-titre "Bien vivre la chaleur" (vs "il fait chaud")

| Décision                                                                                                             | Pourquoi                                                                                                                                                                                                                                                                                                                                         | Alternatives considérées                                                                    | Statut |
| -------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------- | ------ |
| "Ifecho — Bien vivre la chaleur" remplace "Ifecho - il fait chaud" dans `manifest.json`, `index.html` et `CLAUDE.md` | "il fait chaud" explique le jeu de mots phonétique dans le sous-titre — ce qui le tue — ET gaspille l'espace sans décrire la valeur produit. "Bien vivre la chaleur" couvre V0 (ventilation) + V1 (alertes canicule, conseils survie) sans s'enfermer sur une seule feature. Débat Rodin : un sous-titre décrit la valeur, ne décode pas le nom. | "Quand ouvrir vos fenêtres ?" (trop V0, trop étroit) ; "Conseils canicule" (trop alarmiste) | actif  |
