# BDR-006 — API Adresse gouvernement pour autocomplete commune

| Décision                                                                                             | Pourquoi                                                                                                                                                | Alternatives considérées                                                                            | Statut |
| ---------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- | ------ |
| `api-adresse.data.gouv.fr` avec paramètre `type=municipality` pour la saisie de la ville utilisateur | Gratuit, sans clé API, officiel, retourne lat/lon de la commune. `type=municipality` filtre sur les communes uniquement (pas les rues ni les adresses). | Nominatim (pas de filtre municipality natif), Google Places API (payant), Algolia Places (déprécié) | actif  |
