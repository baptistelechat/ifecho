# BDR-029 — `uvPenalty` remplace `bonusNight` dans le score ventilation

| Décision                                                                                                                               | Pourquoi                                                                                                                                                                                                                             | Alternatives considérées                                                                        | Statut |
| -------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- | ------ |
| `score = deltaT - uvPenalty(uvIndex)` remplace `score = deltaT + bonusNight` · seuils : uvIndex ≥ 6 → 1.5, ≥ 3 → 1, ≥ 1 → 0.5, sinon 0 | `apparent_temperature` Open-Meteo exclut le rayonnement solaire (vent + humidité seulement) — `uvPenalty` via `uvIndex` (déjà dans le payload) comble ce gap physiquement : fort UV = gain solaire par les vitres pendant l'aération | `bonusNight` fixe +1 (grossier, binaire jour/nuit) ; flag `isNight` (cliff artificiel à minuit) | actif  |
