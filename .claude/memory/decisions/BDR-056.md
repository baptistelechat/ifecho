# BDR-056 — Guard `isLocalhost` dans `main.tsx` pour PostHog

| Décision                                                                                              | Pourquoi                                                                                                                                                  | Alternatives considérées                                           | Statut |
| ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ | ------ |
| `window.location.hostname === 'localhost' \|\| '127.0.0.1'` ajouté à `posthogEnabled` dans `main.tsx` | `pnpm preview` sert le build prod Vite → `PROD=true` sur localhost → PostHog actif sans ce guard. A généré ~350 events de test orphelins non supprimables | Env var `VITE_POSTHOG_DISABLE=true` (rejeté — oubliable, manuelle) | actif  |

## Références

- voir aussi GLRN-153 — `pnpm preview` → `PROD=true` sur localhost (global)
