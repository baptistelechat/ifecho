# BDR-056 — Guard `isLocalhost` dans `main.tsx` pour PostHog

| Décision                                                                                 | Pourquoi                                                                                                                                                                                                             | Alternatives considérées                                                                                         | Statut |
| ---------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------ |
| `hostname === 'localhost' \|\| /^\d{1,3}(\.\d{1,3}){3}$/.test(hostname)` dans `main.tsx` | `pnpm preview` sert le build prod Vite → `PROD=true` → PostHog actif sans ce guard. Regex IP couvre aussi les tests téléphone via adresse LAN (192.168.x.x). A généré ~350 events de test orphelins non supprimables | Env var `VITE_POSTHOG_DISABLE=true` (rejeté — oubliable, manuelle) ; check `127.0.0.1` seul (incomplet pour LAN) | actif  |
