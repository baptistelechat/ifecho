# BDR-011 — Layout App.tsx : flex-col sémantique + footer collé en bas

| Décision                                                                                                                 | Pourquoi                                                                                          | Alternatives considérées                                                         | Statut |
| ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- | ------ |
| `min-h-dvh flex flex-col` avec `<header>` / `<main flex-1>` / `<footer>` ; `my-auto` conditionnel sur le contenu interne | Footer devait être toujours en bas de page, champ de recherche centré verticalement à l'état vide | Footer dans le flux du contenu (avant) — flottait au milieu quand peu de contenu | actif  |

Le conteneur interne du `<main>` reçoit `my-auto` uniquement quand `!hasContent` (pas de météo chargée). Quand les données arrivent, `my-auto` disparaît et le contenu s'aligne en haut naturellement. Footer avec signature "Made by @baptistelechat with Heart Lucide" + lien portfolio.
