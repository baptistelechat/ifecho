# BDR-047 — Wrapper absolu header : `ShareButton` + `InstallButton` ensemble

| Décision                                                                                                                                  | Pourquoi                                                                                                                                                   | Alternatives considérées                                                                                        | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- | ------ |
| `div.absolute.right-4.top-4.flex.items-center.gap-2` dans App.tsx contient ShareButton + InstallButton. InstallButton devient `relative`. | InstallButton avait son propre `absolute top-4 right-4` incompatible avec layout côte à côte dans un flex. Responsabilité de placement remontée au parent. | Garder le absolute dans InstallButton + ajouter un wrapper externe, ou absolute indépendants pour chaque bouton | actif  |
