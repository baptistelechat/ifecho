# LRN-039 — `pnpm add` workspace root exige le flag `-w`

| Pattern observé                                                                                    | Contexte                                                                                                                                                              | Application future                                                                                                                                          |
| -------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| En présence d'un `pnpm-workspace.yaml`, `pnpm add` sans `-w` échoue avec `ERR_PNPM_ADDING_TO_ROOT` | Installation de `vite-plugin-pwa` et `@vite-pwa/assets-generator` dans un projet single-package avec `pnpm-workspace.yaml` présent (créé pour hardening supply-chain) | Toujours utiliser `pnpm add -D -w <package>` pour ajouter dans le `package.json` racine d'un workspace, même si ce workspace ne contient qu'un seul package |
