# LRN-064 — `lucide-animated` : div wrappers `AnimHandle` ≠ `LucideIcon` → cast + `LoopIcon`

| Pattern observé                                                                                                                                                                                                               | Contexte                                                                                    | Application future                                                                                                                                                                                                                                              |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Les composants lucide-animated sont des `ForwardRefExoticComponent<HTMLAttributes<HTMLDivElement> & { size?: number } & RefAttributes<AnimHandle>>` — incompatibles avec `LucideIcon` (SVG). Animation hover-only par défaut. | Intégration `SunIcon`, `CloudRainIcon`, etc. dans `WeatherIcon.tsx` — les animer sans hover | Cast `as unknown as AnimIcon` pour typer uniformément. Pattern autonome : `useRef<AnimHandle>` + `useEffect` cycle `startAnimation → (1600ms) → stopAnimation → (2500ms) → repeat`. Icônes sans équivalent animé → fallback `lucide-react` statiques sans cast. |
