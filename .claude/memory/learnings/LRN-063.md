# LRN-063 — `gap-px bg-border` grid CSS → contenu flush aux bords de card

| Pattern observé                                                                                                                                                                                                       | Contexte                                                                                                                   | Application future                                                                                                                                                                 |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `grid gap-px` avec `bg-border` sur le conteneur crée les séparateurs en laissant transparaître le fond — mais les cellules s'étendent jusqu'aux bords absolus du parent, donnant l'impression de zéro padding latéral | `VigilanceBanner` : le grid s'étendait d'un bord à l'autre de la card `overflow-hidden`, icônes et labels au ras des bords | Tout grid Tailwind dans une card : préférer un wrapper `px-N` autour du grid + `border-r border-border` sur les cellules de colonne gauche + `border-b` sur les lignes non-finales |
