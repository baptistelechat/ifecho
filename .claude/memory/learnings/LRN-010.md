# LRN-010 — PowerShell `@'...'@` perd accents/° → ASCII + Edit `replace_all: true`

| Pattern observé                                                                                                                                                                      | Contexte                                                                                                                                                              | Application future                                                                                                                      |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Écrire un fichier accentué via PowerShell : (1) Write avec substituts ASCII (ex: `e` pour `é`, `deg` pour `°`), (2) Edit séquentiels `replace_all: true` pour restaurer chaque token | Python via PowerShell bloqué par sandbox (Remove-Item sur `$env:TEMP`). `@'...'@` perd silencieusement les accents. Biome hook invalide la lecture entre chaque Write | Tout projet Windows + Biome + fichiers TS contenant des strings accentués → toujours Write ASCII first, puis Edit ciblés pour restaurer |
