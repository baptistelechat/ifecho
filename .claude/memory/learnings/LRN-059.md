# LRN-059 — Bulletin météo multi-phénomènes : péremption = niveau bulletin, pas item

| Pattern observé                                                                                                                                                                                                                                                                           | Contexte                                                                                                                                                                                                                        | Application future                                                                                                                                       |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Plusieurs phénomènes partagent le même bulletin émis au même instant. Un phénomène sans item J (vert la veille = 0 enregistrement) n'est PAS frais — son absence d'item est l'absence de vigilance hier, pas un "bulletin ok". La staleness doit être agrégée sur l'ensemble du bulletin. | VigilanceBanner après minuit : Canicule avait des items J expirés → détection ok. Orages avait `j=[]` (vert vendredi) → per-phénomène ratait le swap, icône restait verte alors que samedi = jaune. Fix : bulletinStale global. | Toute API retournant un bulletin groupé (N phénomènes/catégories émis ensemble) : contrôler la fraîcheur au niveau du bulletin entier, pas item par item |
