# LRN-021 — Icônes SVG complexes (Lucide) illisibles < 12px → composite base + modificateur

| Pattern observé                                                                                                                                                                                  | Contexte                                                                                        | Application future                                                                                                                                          |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Sunrise`/`Sunset` ont plusieurs paths SVG — à `size-3` (12px) ils deviennent un blob illisible. Fix : `Sun` + `ArrowUp`/`ArrowDown` côte à côte donnent le même sens avec une lecture immédiate | Pictos "Matin frais" (↑) et "Soirée" (↓) dans `IdealSlots` — signalé par screenshot utilisateur | Toute icône Lucide comportant 3+ paths SVG : tester le rendu à `size-3`. Si illisible → décomposer en icône de base universelle + modificateur directionnel |
