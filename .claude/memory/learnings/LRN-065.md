# LRN-065 — `forecast_days: "3"` = 72h, ms-window peut inclure J+2

| Pattern observé                                                                                                                                                                                      | Contexte                                                                                    | Application future                                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| `forecast_days: "3"` retourne 72h de prévisions. Une fenêtre ms (ex. +28h depuis samedi soir) peut donc inclure des heures de J+2 (lundi). Toujours filtrer par date string plutôt que par offset ms | Ifecho affichait des horaires du lundi un samedi soir — invisible en dev (testé en journée) | Avec Open-Meteo : `forecast_days: "2"` + filtre `date === todayDate \|\| date === tomorrowDate` |
