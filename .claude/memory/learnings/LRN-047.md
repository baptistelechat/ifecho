# LRN-047 — `persistence: 'memory'` PostHog → events orphelins, bulk-delete inefficace

| Pattern observé                                                                                                                                                                                                                                      | Contexte                                                                 | Application future                                                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| Avec `persistence: 'memory'`, les events sont capturés sans person profile → `persons-bulk-delete` ne touche pas ces events. Seul un filtre sur une propriété de l'event (`$host`, `$current_url`) via `test_account_filters` permet de les exclure. | 229 events localhost:5173 impossibles à supprimer post-facto dans ifecho | Prévoir dès le départ un filtre `$host` pour distinguer dev/prod. Ne pas compter sur la suppression post-facto. |
