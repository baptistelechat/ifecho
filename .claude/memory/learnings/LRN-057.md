# LRN-057 — ESLint global vs local → TypeError sur rules @typescript-eslint

| Pattern observé                                                                                                                                                                                                                                        | Contexte                                               | Application future                                                                                                                     |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| `pnpm exec eslint` utilise le binaire global (9.9.0) qui charge les rules du local (9.39.4) → `TypeError: Cannot read properties of undefined (reading 'allowShortCircuit')`. Fix : `pnpm lint` via scripts package.json pour forcer le binaire local. | ifecho sur Windows — `pnpm exec eslint` vs `pnpm lint` | Tout projet Node avec ESLint installé localement : toujours passer par `pnpm lint` (script), jamais par `pnpm exec eslint` directement |
