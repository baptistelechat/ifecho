# LRN-062 — Fin de créneau +1h/heure = fenêtre de données glissante

| Pattern observé                                                                                                  | Contexte                                                                                                                                                                       | Application future                                                                                                                         |
| ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Un horodatage de fin qui se décale de +1h à chaque heure écoulée = troncation données upstream, pas météo réelle | À 17h04 : fin = "17h". `IdealSlots` (full data) ET timeline (slice 25) montraient le même horodatage — la cause était dans le hook `useVentilationScore`, pas dans l'affichage | Si deux composants avec des sources différentes affichent la même fin suspecte : chercher dans le hook source avant de toucher l'affichage |
