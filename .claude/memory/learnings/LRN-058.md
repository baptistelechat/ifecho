# LRN-058 — Supprimer valeurs numériques des messages si affichées par un frère

| Pattern observé                                                                                           | Contexte                                                                                                                         | Application future                                                                                                                   |
| --------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Un composant message ne doit pas répéter une valeur numérique qu'un composant frère affiche déjà en clair | Messages de `ThermalDelta` répétaient `${delta.toFixed(1)}°C` alors que `ThermalComparison` l'affiche en grand dans la même card | Avant d'inclure une valeur numérique dans un message textuel, vérifier si le layout en affiche une instance lisible dans la même vue |
