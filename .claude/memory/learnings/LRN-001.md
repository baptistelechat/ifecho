# LRN-001 — Algo score ventilation : ΔT + malus_humidité + bonus_heure

| Pattern observé                             | Contexte                              | Application future                                              |
| ------------------------------------------- | ------------------------------------- | --------------------------------------------------------------- |
| Score horaire = ΔT + malus_humidité + bonus | Calcul client avec données Open-Meteo | Réutilisable en V2 avec variables supplémentaires (AQI, pollen) |

## Formule

```
score(heure) = ΔT + malus_humidité + bonus_heure
```

| Variable         | Définition                         | Valeur                                             |
| ---------------- | ---------------------------------- | -------------------------------------------------- |
| `ΔT`             | `temp_int - temp_ext(heure)` en °C | continu — plus élevé = meilleur moment pour ouvrir |
| `malus_humidité` | si `humidité_ext(heure) > 80%`     | `-3` (ouvrir humidifie le logement) sinon `0`      |
| `bonus_heure`    | si `heure ∈ [sunset, sunrise]`     | `+2` (créneau nocturne réel) sinon `0`             |

**Seuil minimal** : `score > 0` pour qu'une heure soit favorable. Le pic de score sur 24h = l'heure cible recommandée.

## Données Open-Meteo nécessaires

```
GET https://api.open-meteo.com/v1/forecast
  ?latitude={lat}&longitude={lon}
  &hourly=temperature_2m,relativehumidity_2m
  &daily=sunrise,sunset
  &forecast_days=1
  &timezone=Europe/Paris
```

Retourne 24 températures + 24 humidités + lever/coucher du soleil du jour. Sans clé API.
