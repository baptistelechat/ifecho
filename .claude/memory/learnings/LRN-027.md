# LRN-027 — Hook retournant objet instable → event-handler ref

| Pattern observé                                                                                                                                                                                                                                                                                         | Contexte                                                                                                                                       | Application future                                                                       |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- |
| Quand un hook tiers retourne un nouvel objet à chaque render, le mettre dans les deps `useEffect` cause une relance infinie. Fix : `const methodRef = useRef(hook.method); methodRef.current = hook.method;` puis appeler `methodRef.current()` dans l'effet — la ref est stable, la valeur est fraîche | `useHaptics()` retourne `{ nudge, success, error }` sans mémoïsation — inclure `haptics` dans les deps déclenchait une relance à chaque render | Appliquer dès qu'un hook tiers retourne un objet non mémoïsé utilisé dans un `useEffect` |
