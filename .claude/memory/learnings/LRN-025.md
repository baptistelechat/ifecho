# LRN-025 — `setPointerCapture` pour capturer drag events hors bounds

| Pattern observé                                                                                                                                                | Contexte                                                                            | Application future                                                                                                  |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Sans `setPointerCapture`, si le curseur/doigt quitte l'élément avant `pointerup`, les events pointer sont perdus — le drag/swipe se "décroche" silencieusement | Carousel TipsSection : fin de swipe hors du container non détectée, delta incorrect | Appeler `e.currentTarget.setPointerCapture(e.pointerId)` dans `onPointerDown` pour tout composant drag/swipe custom |
