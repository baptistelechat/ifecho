# LRN-002 — Règle des 500 W/m² — fenêtre standard = radiateur 500 W

| Pattern observé                                   | Contexte                                    | Application future                       |
| ------------------------------------------------- | ------------------------------------------- | ---------------------------------------- |
| Vitrage 4-16-4 standard transmet ~500 W/m² en été | Feature conseils vitrage (contenu statique) | Cartes informatives TipsSection de l'app |

## Formule

```
Puissance transmise (W) = Rayonnement solaire (W/m²) × Facteur solaire (g) × Surface (m²)
```

Rayonnement au sol en journée ensoleillée : ~900-1000 W/m². Facteur g d'un vitrage 4-16-4 : ~0,70. Résultat : **≈ 500 W/m²**.

## Tableau de référence

| Ouvrant          | Surface | Puissance transmise |
| ---------------- | ------- | ------------------- |
| Fenêtre standard | ~1 m²   | ~500 W              |
| Porte-fenêtre    | ~2 m²   | ~1 000 W            |
| Baie vitrée      | ~4,4 m² | ~2 200 W            |

## Points clés pour les cartes conseils

1. Fermer volets/stores extérieurs **avant 10h** (pas après — la chaleur est déjà entrée)
2. Store/rideau intérieur quasi inutile (rayonnement a déjà traversé le verre)
3. Protections extérieures = seules vraiment efficaces (volets ⭐⭐⭐⭐⭐)
