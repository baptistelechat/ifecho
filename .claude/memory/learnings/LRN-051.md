# LRN-051 — `useOnceVisible` : IntersectionObserver one-shot + `firedRef` pour analytics de première vue

| Pattern observé                                                                                                                                                                                                                                                                                                                                                                      | Contexte                                                                                                                                                            | Application future                                                                                                                                                                                                          |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Hook React qui monte un `IntersectionObserver` sur l'élément via `ref.current`, fire le callback une seule fois à la première intersection (guard `firedRef.current = true`), puis `observer.disconnect()`. Double-guard (`firedRef` dans le setup ET dans le callback) protège contre les edge cases de mount/remount React StrictMode. Retourne le `ref` à attacher au nœud cible. | Tracking `section_viewed` sur `TipsSection` (ifecho) : analytics de première vue sans doublons, sans écoute scroll. `threshold: 0.5` = 50% visible pour déclencher. | Pour tout event analytics "première vue d'un composant". Plus précis que `useEffect + scroll`, élimine les doublons via `firedRef`, disconnect propre = pas de fuite mémoire. Le `threshold` est paramétrable (défaut 0.5). |
