# LRN-015 — React `rules-of-hooks` : tous les hooks AVANT tout `return` conditionnel

| Pattern observé                                                                                                                                                | Contexte                                                                                             | Application future                                                                                                                  |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| ESLint `react-hooks/rules-of-hooks` refuse un `useRef` / `useEffect` déclaré après un `if (...) return null` même si le hook n'est pas utilisé avant ce return | VentilationTimeline avait `useRef + useEffect` après le guard `if (scores.length === 0) return null` | Placer tous les hooks en tête de composant, avant tout early return / guard conditionnel — pas seulement les utiliser, les déclarer |
