# LRN-042 — Lighthouse CLI EPERM Windows → Node API contourne le problème

| Pattern observé                                                                                                                                                                                        | Contexte                                                         | Application future                                                                                                                      |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `npx lighthouse` plante avec EPERM sur Windows (droits socket Chrome). Solution : script Node avec `chrome-launcher` + `lighthouse` npm — même API, mêmes résultats, sans les problèmes de permissions | Script `scripts/lighthouse-audit.mjs` dans ifecho sur Windows 11 | Pour tout projet nécessitant Lighthouse automatisé sur Windows : utiliser Node API (`chrome-launcher` + `lighthouse`) plutôt que le CLI |
