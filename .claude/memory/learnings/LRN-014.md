# LRN-014 — `scrollIntoView` > `offsetLeft` pour centrage dans un scroll container

| Pattern observé                                                                                                                                      | Contexte                                                                                                               | Application future                                                                                                                                          |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `offsetLeft` = position relative à l'`offsetParent` DOM, pas au scroll container → décalage si un ancêtre `position: relative` existe entre les deux | VentilationTimeline : un descendant `relative` entre le scroll container et la colonne faussait le calcul `offsetLeft` | Toujours préférer `el.scrollIntoView({ inline: 'center', block: 'nearest' })` dans `useEffect(() => {}, [])` pour centrer un élément dans son scroll parent |
