# LRN-067 — État vide diagnostique : checklist ✓/✗ > message générique

| Pattern observé                                                                                                                                                                                     | Contexte                                                                                                                                                | Application future                                                                                                                       |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Pour un état vide résultant de N conditions, afficher une checklist ✓/✗ par condition est plus actionnable qu'un message générique. L'utilisateur comprend immédiatement ce qui bloque et peut agir | IdealSlots ifecho : "Aucun créneau favorable" sans explication. Checklist 3 conditions (indoor / score / outdoor) permet le diagnostic en un coup d'œil | Tout composant filtrant des données avec N conditions : si résultat vide, afficher les conditions avec leur statut actuel `met: boolean` |
