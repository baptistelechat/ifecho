# LRN-037 — `resizeOptions.background` contrôle la couleur de fond du padding PWA

| Pattern observé                                                                                                                                                                   | Contexte                                                                                                                                      | Application future                                                                                                                       |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Pour remplacer le fond blanc par défaut du preset, ajouter `resizeOptions: { background: "#hex" }` dans les sections `maskable` et `apple` du config `@vite-pwa/assets-generator` | `minimal2023Preset` utilise blanc pour le padding des icônes maskable (W3C safe zone 10%) et apple-touch-icon (30%) — visible sur fond coloré | Toujours personnaliser `resizeOptions.background` à la valeur de `theme_color` du manifest pour cohérence visuelle sur l'écran d'accueil |
