# LRN-035 — Label contextuel : afficher seulement quand il lève une ambiguïté

| Pattern observé                                                                                                                                                                      | Contexte                                                                                                                                                           | Application future                                                                                                                         |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Absence d'un label = information implicite. Présence = signal actif. Afficher uniquement quand le contexte est ambigu — ici, quand des créneaux de 2 jours coexistent dans la liste. | `IdealSlots.tsx` : `• Demain` ajouté d'abord, puis symétrisé avec `• Aujourd'hui` uniquement si mix J/J+1 détecté via `slots.some(s => s.startDate !== todayStr)`. | Tout badge, date, tag ou indicateur d'état : "ce label est-il visible uniquement quand il différencie quelque chose ?" Si non → supprimer. |
