# LRN-056 — ODS N records pour un même phénomène = transition de niveau

| Pattern observé                                                                                                                                                                                                                                              | Contexte                                                       | Application future                                                                                                                              |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| API ODS vigilance retourne N enregistrements pour le même phénomène quand il y a une transition de niveau sur la même échéance (ex : canicule orange puis canicule jaune sur J+1). Grouper par `phenomenon` + trier par `begin_time` pour détecter et gérer. | VigilanceBanner, dept 85, transition J+1 canicule orange→jaune | Toute intégration d'API alerte publique : ne pas supposer 1 record par phénomène/période. Grouper par identité phénomène, trier temporellement. |
