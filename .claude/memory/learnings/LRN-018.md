# LRN-018 — Chart bidirectionnel : `score === 0` → colonne invisible → `MIN_BAR`

| Pattern observé                                                                                                                                        | Contexte                                                                                                                              | Application future                                                                                                                                   |
| ------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `score === 0` mappé à `(0 / maxAbs) * ZONE_HEIGHT = 0px` → la colonne disparaît de la timeline, créant un gap visuel entre les barres vertes et rouges | Heures où `deltaT = 0` et `bonusNight = 0` (milieu de journée, même temp int/ext) n'affichaient aucune barre dans VentilationTimeline | Tout chart bidirectionnel : `s > 0 ? Math.max(s/max * ZONE, MIN_BAR) : s === 0 ? MIN_BAR : 0` pour éviter les colonnes invisibles sur valeurs nulles |
