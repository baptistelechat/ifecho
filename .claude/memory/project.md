# URL de production — Ifecho

**URL** : https://ifecho.vercel.app

**Why:** Déployée sur Vercel (branche `main`), SPA statique 100% client-side. Aucune variable d'environnement requise.

**How to apply:** Utiliser cette URL dans tous les contextes prod : README badges, EPIC docs, partage externe, tests Lighthouse, tests PWA. Pour tester la PWA, désactiver temporairement Vercel Authentication (voir [[BLK-029]]).
